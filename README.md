**🎲 Real-Time 2-Player Bingo Game**

A simple real-time multiplayer bingo game built with HTML, CSS, JavaScript, Node.js, and Express.js. It supports LAN multiplayer, live chat, turn-based play, dark mode, and mobile-friendly design.

**Features**
- 2-player real-time bingo
- LAN multiplayer
- Live chat
- Turn-based gameplay
- Dark mode
- Mobile responsive
- New Game reset
- Clear Chat option

**Technologies Used**
- HTML
- CSS
- JavaScript
- Node.js
- Express.js

**Installation**
1. Install Node.js LTS.
2. Open the project folder(LAN-BINGO) in terminal.
3. Run: npm install
4. Run: node server.js or npm start

**Open the Game**

**Host:** http://YOUR-IP:8080

**Guest:** http://YOUR-IP:8080?type=guest


**How to Play**
1. Host starts first.
2. Click a number on your turn.
3. The same number will mark on both boards.
4. First player to complete 5 lines wins.

**Important**
- Both devices must be on the same WiFi OR LAN.
- Use the correct IP address.
- Keep the server running while playing.

**Common Fixes**
- If npm is not recognized, restart PC and reinstall Node.js.
- If the game does not open on mobile, check WiFi and IP address.
- To stop the server, press CTRL + C.
